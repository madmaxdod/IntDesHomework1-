Web Search Homework

Findings from my digital "sleuthing" on two websites.

Screenslate.com Analysis

My first website was screenslate.com. After using the Chrome Inspector and looking for clues, I discovered that this website was built using React.js.

While the absence of obvious script tags for React made me think it used a hand-coded solution, but digging deeper into the source tab I was able to find certain elements using the .react tag that leads me to believe it is using react.js. The project is likely built on a more subtle or custom implementation of a React framework, which still relies on core web technologies like plain HTML and CSS, but uses a library to handle component-based UI.

This website, although I love its functionality, became overwhelming pretty quickly, so I wanted to move to something more digestable.

Clicking.toys Analysis

For my second website, I analyzed clicking.toys. This site was a perfect example of a different type of web project. I found a <canvas> element in the HTML, which suggested the presence of generative art.

By inspecting the code, I confirmed that the site is also not using a major framework. The game itself is powered by vanilla JavaScript, with the core game logic and scoring handled in custom files named flip-grid.min.js and scoring.min.js. The use of the <canvas> element and a script file points towards a library for creative coding called p5.js being used. The game's interactive "flipping" visuals are handled by HTML div elements, while the canvas likely provides background effects or a secondary layer.

These simple java files were minified, so the full logic was a bit confusing.

I looked into the CSS file and found a media query function for the game that activates at 850px. This code is telling the browser to apply specific style changes, to move the header and extend the text across a banner on top of the screen, only when the width is 850 pixels or less. The purpose of this is to make the website look good and be easy to use on a mobile device. For example, it changes the layout from side-by-side to a single column and hides some elements like ads to save visual space.
